# Case de Implantação ERP

Projeto demonstrativo de implantação de um sistema ERP para a empresa fictícia **Distribuidora Nordeste Ltda.**, visando a substituição de processos manuais por uma gestão integrada e automatizada.

## 🏢 Cenário da Empresa
- **Empresa:** Distribuidora Nordeste Ltda.
- **Segmento:** Distribuição de Alimentos.
- **Dores Atuais:** Controle de estoque em planilhas, compras manuais, falta de integração financeira e ausência de indicadores gerenciais.

## 🎯 Objetivos do Projeto
- Integrar processos entre setores.
- Automatizar o setor de compras.
- Garantir a confiabilidade e rastreabilidade do estoque.
- Melhorar a gestão e o fluxo de caixa financeiro.

## 🧭 Metodologia de Implantação
1. Levantamento de Requisitos
2. Cronograma e Escopo
3. Parametrização do Sistema
4. Migração de Dados
5. Treinamento de Usuários
6. Plano de Go-Live
7. Pós Go-Live e Acompanhamento
